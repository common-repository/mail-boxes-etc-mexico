Integracion de MBE Shipping de Mail Boxes Etc. a Woocommerce.

== Description ==

¿Estás buscando un proveedor que se encargue de la operación logística de tus productos?

Mail Boxes etc. Es la mejor compañía en soluciones logisticas empresariales. Analizamos tu negocio, te asesoramos ofreciendo las mejores opciones de envío, adaptándonos a tus necesidades y generando importantes ahorros operativos.  

Dentro de nuestra plataforma podrás comparar los diferentes servicios de las mejores mensajerías del país, dar seguimiento en tiempo real a tus paquetes y consultar el estatus de cada uno de ellos, entre otros.  Nos encargamos de todas las actividades de la cadena logística, desde recolecciones, pick and pack, empaque personalizado y más.

== Installation ==

1. Upload 'Mail-boxes-mex' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on WooCommerce > Settings
4. Click on MBE Shipping Tag and set your store information, which will be provided by MBE Shipping de Mail Boxes Etc.
